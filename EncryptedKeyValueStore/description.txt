KeyValue store whose keys and values are cryptograhpically encrypted with secret key.

New-EKVStore.ps1
+ store name
+ master password of .ekv file
+ returns store file path
-> creates a new file with given name, .ekv extension and in location ./ekvs/
-> first line of .ekv file is a SHA256 hash of master password
-> other lines are:
    1. key
    2. encrypted value

Add-EKVRecord.ps1
+ store name
+ store password
+ key value
+ value value
-> checks whether hash of given password is equal to first line
-> if not equal does nothing
-> if equal, encrypts value with given password as secret key and makes a new record in .ekv file

Get-EKVRecord.ps1
+ store name
+ store password
+ key value
-> checks whether hash of given password is equal to first line
-> if not equal does nothing
-> if equal, decrypts value with given password as secret key under given key and returns value

Remove-EKVRecord.ps1
+ store name
+ store password
+ key value
-> checks whether hash of given password is equal to first line
-> if not equal does nothing
-> if equal, removes the record under given key from .ekv file

Remove-EKVStore.ps1
+ store name
+ store password
-> check whether .ekv file exists
-> check whether hash of given password is equal to first line
-> if not equal does nothing
-> if equal, removes .ekv file
